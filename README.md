# 🔬 Vulnerability Research & Security Lab

Este repositório hospeda o conteúdo do meu blog técnico: [lucazssilva.github.io](https://lucazssilva.github.io/).

### 🎯 Objetivo
O propósito deste projeto é centralizar a documentação dos meus estudos em **Offensive Security** e **AppSec**. Utilizo este espaço para:

* **Write-ups:** Documentação detalhada de máquinas, desafios de CTF.
* **Vulnerability Research:** Análise de vulnerabilidades reais e estudos sobre o OWASP Top 10.
* **Automação:** Scripts em Python e Bash voltados para tarefas de reconhecimento e exploração.
* **PoCs (Proof of Concept):** Demonstração técnica de vulnerabilidades identificadas durante meus estudos.

### 🛠️ Tech Stack
* **Static Site Generator:** Jekyll 
* **Hospedagem:** GitHub Pages
* **Linguagem:** Markdown
